# PROJECT OVERVIEW

## What This Site Is

A standalone proposal website built to present a strategic response to the SPARK Schools RFP. It communicates our approach, methodology, team, and value proposition in a polished, brand-aligned format designed to support the decision-making process.

## Target Audience

- SPARK Schools leadership team
- Decision-makers reviewing RFP submissions
- Internal stakeholders evaluating vendor proposals

## Deployment Environment

- **Platform**: Vercel
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4 + shadcn UI
- **Language**: TypeScript
